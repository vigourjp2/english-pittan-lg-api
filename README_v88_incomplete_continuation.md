# v88 incomplete continuation fix

目的: 英文を作っている途中の `I` / `I like` / `I am` などを、即「不成立・罰点」にしない。

修正:
- 成立1件以上: v87方針どおりNG経路は完全無視。
- 成立0件: すぐ罰点にせず、代表候補/TOP行について手札1〜2枚を後ろに足した候補をAPI batchで確認。
- 補完候補が成立する場合: `採点前` と表示し、罰点なしで盤面に残す。
- 補完候補が見つからない場合だけ: TOP1 NG経路だけ理由解析し、罰点。
- 例: `I`、`I like`、`I am` は続ければ成立しうるため罰点なし。`I like apples` になった時点で採点。

判定方針:
- 文法ハードコードで成立判定しない。
- 補完可能性チェックも `/check-and-translate-batch` を使い、APIの `gameOk:true` だけを信用する。
